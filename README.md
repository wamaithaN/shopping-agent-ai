# shopping-agent-ai
An AI-powered shopping assistant that helps users search, compare, and recommend products using agentic AI workflows.
This project demonstrates an autonomous shopping assistant capable of:

searching products
comparing options
recommending products
answering shopping-related questions
planning shopping decisions using AI agents

Features
Product search
Product recommendation
Intelligent comparison
Multi-step reasoning
Agent-based workflow
Easy-to-use interface

Tech Stack
Python
OpenAI API 
LangChain 
Streamlit 
Pandas
