# 🛒 Shopping Agent AI

An AI-powered shopping assistant that helps users search, compare, and recommend products using agentic AI workflows.

## 🚀 Overview

Shopping Agent AI is an autonomous AI assistant designed to simplify shopping decisions. It can search for products, compare multiple options, answer shopping-related questions, and recommend products based on user preferences.

## ✨ Features

- 🔍 Product search
- 📊 Product comparison
- 🤖 AI-powered recommendations
- 🧠 Multi-step reasoning using AI agents
- 💬 Shopping question answering
- ⚡ Simple and interactive user interface

## 🛠️ Tech Stack

- Python
- OpenAI API
- LangChain
- Streamlit
- Pandas

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/wamaithaN/shopping-agent-ai.git
```

Navigate to the project folder:

```bash
cd shopping-agent-ai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
